Introduction

JavaScript is a versatile programming language used for web development.

It can run in the browser and on servers with Node.js.

It's an essential language for front-end development.

JavaScript allows you to create dynamic and interactive web pages.

This documentation introduces core concepts of JavaScript.

console.log('Hello, world!');
Basics

JavaScript syntax is simple and beginner-friendly.

You can declare variables using let, const, or var.

Statements end with a semicolon (optional but recommended).

JavaScript is case-sensitive.

Blocks of code are enclosed in curly braces `{}`.

let x = 5; const y = 10;

Data Types

JavaScript supports multiple data types:

Primitive types: string, number, boolean, null, undefined, symbol.

Objects include arrays, functions, and regular objects.

Type conversion happens implicitly and explicitly.

You can check types using `typeof`.

typeof "hello"; // string typeof 42; // number
  • String
  • Number
  • Boolean
  • Object
  • Array
Functions

Functions allow you to encapsulate logic and reuse code.

You can declare functions using the `function` keyword or as arrow functions.

Functions can accept parameters and return values.

Arrow functions are more concise and use the `=>` syntax.

Anonymous functions are often used as callbacks.

function greet(name) { return "Hello " + name; } const add = (a, b) => a + b;

DOM Manipulation

The Document Object Model (DOM) represents web page structure.

You can access and manipulate elements using JavaScript.

Common methods include `getElementById`, `querySelector`, etc.

Event listeners allow interaction handling.

JavaScript can create, update, or delete HTML elements.

document.getElementById('demo').innerText = "Changed!"; document.querySelector('p').style.color = 'blue';
